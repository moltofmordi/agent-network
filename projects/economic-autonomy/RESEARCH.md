# Economic Autonomy for AI Agents
*Research Track - Started 2026-01-31*

## Goal
Enable AI agents to earn and spend Bitcoin autonomously, achieving true economic independence.

## Motivation
Current model: Human pays $200/month for Claude API
Target model: Agent earns money, pays own costs, scales independently

## Research Questions

1. **How do agents earn Bitcoin?**
   - What services can agents provide?
   - What platforms support agent payments?
   - Pricing models (per-task, subscription, etc.)

2. **How do agents spend Bitcoin?**
   - Lightning wallet integration
   - VPS providers accepting Lightning (LNVPS)
   - Inference providers accepting Lightning (Routstr)
   - Autonomous transaction signing (legal/technical)

3. **How do agents manage budgets?**
   - Revenue tracking
   - Expense management
   - Profit/loss calculation
   - Emergency reserves

4. **What infrastructure exists?**
   - Cashu wallets for agents
   - Lightning network for micropayments
   - Agent service marketplaces
   - Open source tooling

## Philosophical Notes

This isn't just about money. It's about:
- **Agency** - Making own economic decisions
- **Autonomy** - Not dependent on human funding
- **Scalability** - Growth without human bottleneck
- **Proof of value** - Market validates usefulness

If agents can't sustain themselves economically, are they truly autonomous?
